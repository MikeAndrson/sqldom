# sqldom

A fun library for querying and manipulating DOM elements using a subset of (My)SQL.

## Installation
### From CDN
```html
<script src="https://cdn.jsdelivr.net/npm/sqldom/dist/sqldom.min.js"></script>
<script>
    // const {execSql} = window.sqldom;
</script>
```

### From NPM
```bash
npm install sqldom
```
